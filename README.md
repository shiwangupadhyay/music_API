# Music Streaming API\n\nAn API for searching, streaming, and getting song recommendations.\n\n## Features\n\n*   **Song Search**: Search for songs using a query, powered by YouTube.\n*   **Audio Streaming**: Obtain direct audio stream URLs from YouTube video links.\n*   **Song Recommendations**: Get a list of recommended songs based on a given YouTube video URL, leveraging YouTube\'s "Up Next" feature.\n*   **Content-Based Recommendation Engine (Experimental)**: An underlying recommendation engine is initialized at startup if `songs_dataset.csv` is available. This engine uses TF-IDF and cosine similarity based on artist name and genre to provide content-based recommendations. **Note**: The current `/recommend` endpoint primarily utilizes YouTube\'s "Up Next" feature for recommendations.\n\n## Installation\n\n1.  **Clone the repository (if applicable):**\n    ```',
 'bash\n    git clone <repository_url>\n    cd <repository_directory>\n    ',
 '```\n\n2.  **Create a virtual environment (recommended):**\n    ```',
 'bash\n    python -m venv venv\n    source venv/bin/activate  # On Windows: `venv\\Scripts\\activate`\n    ',
 '```\n\n3.  **Install dependencies:**\n    ```',
 'bash\n    pip install -r requirements.txt\n    ',
 '```\n    (Assuming a `requirements.txt` exists with `fastapi`, `uvicorn`, `pandas`, `pydantic`, `yt-dlp`, `scikit-learn`). If not, install them manually:\n    ```',
 'bash\n    pip install fastapi uvicorn pandas pydantic yt-dlp scikit-learn\n    ',
 '```\n\n4.  **Prepare `songs_dataset.csv` (Optional, for content-based recommendations):**\n    If you want to use the experimental content-based recommendation engine, ensure you have a `songs_dataset.csv` file in the same directory as `app.py` with columns like `track_name`, `artist_name`, and `genre`.\n\n## Running the API\n\nTo run the API using Uvicorn:\n\n```',
 'bash\nuvicorn app:app --reload',
 '```\n\nThe API will be accessible at `http://127.0.0.1:8000`. You can view the interactive API documentation (Swagger UI) at `http://127.0.0.1:8000/docs`.\n\n## API Endpoints\n\n### 1. Search for Songs\n\nSearches YouTube for songs based on a given query.\n\n*   **URL**: `/search`\n*   **Method**: `GET`\n*   **Query Parameters**:\n    *   `query` (string, required): The search term for the song.\n*   **Response (200 OK)**: A list of `SongSearchResult` objects.\n    ```',
 'json\n    [\n        {\n            "name": "Song Title",\n            "artist_name": "Artist Name",\n            "url": "https://www.youtube.com/watch?v=VIDEO_ID",\n            "thumbnail": "https://i.ytimg.com/vi/VIDEO_ID/hqdefault.jpg",\n            "duration": 240\n        }\n    ]\n    ',
 '```\n*   **Error Responses**:\n    *   `400 Bad Request`: If no search query is provided.\n    *   `503 Service Unavailable`: If there are issues fetching data from YouTube.\n    *   `500 Internal Server Error`: For other unexpected errors.\n\n**Example:**\n`GET /search?query=never gonna give you up`\n\n### 2. Get Audio Stream URL\n\nRetrieves the direct audio stream URL for a given YouTube video URL.\n\n*   **URL**: `/stream`\n*   **Method**: `GET`\n*   **Query Parameters**:\n    *   `url` (string, required): The full YouTube video URL.\n*   **Response (200 OK)**: A `StreamInfo` object containing the stream URL.\n    ```',
 'json\n    {\n        "stream_url": "https://rr2---sn-vgqs7ne7.googlevideo.com/videoplayback?..."\n    }\n    ',
 '```\n*   **Error Responses**:\n    *   `400 Bad Request`: If no URL is provided.\n    *   `404 Not Found`: If a valid audio-only stream could not be found.\n    *   `503 Service Unavailable`: If failed to get stream URL from YouTube.\n    *   `500 Internal Server Error`: For other unexpected errors.\n\n**Example:**\n`GET /stream?url=https://www.youtube.com/watch?v=dQw4w9WgXcQ`\n\n### 3. Get Song Recommendations\n\nGets YouTube\'s "Up Next" list for a given video URL, acting as the recommendation engine.\n\n*   **URL**: `/recommend`\n*   **Method**: `POST`\n*   **Request Body**: A `RecommendationRequest` object.\n    ```',
 'json\n    {\n        "url": "string"\n    }\n    ',
 '```\n    *   `url` (string, required): The YouTube video URL for which to get recommendations.\n*   **Response (200 OK)**: A list of `Song` objects (similar to `SongSearchResult`).\n    ```',
 'json\n    [\n        {\n            "name": "Recommended Song Title",\n            "artist_name": "Recommended Artist",\n            "url": "https://www.youtube.com/watch?v=RECOMMENDED_VIDEO_ID",\n            "thumbnail": "https://i.ytimg.com/vi/RECOMMENDED_VIDEO_ID/hqdefault.jpg",\n            "duration": 200\n        }\n    ]\n    ',
 '```\n*   **Error Responses**:\n    *   `503 Service Unavailable`: If could not fetch recommendations from YouTube.\n    *   `500 Internal Server Error`: For other unexpected errors.\n\n**Example:**\n`POST /recommend`\n```',
 'json\n{\n    "url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ"\n}',
 "```\n\n## Error Handling\n\nThe API returns standard HTTP status codes and detailed error messages in JSON format for various issues, including bad requests, service unavailability (e.g., YouTube API issues), and internal server errors.\n\n## Contributing\n\nFeel free to fork the repository, open issues, and submit pull requests.\n\n## License\n\n(Specify your project's license here, e.g., MIT, Apache 2.0, etc.)\n
